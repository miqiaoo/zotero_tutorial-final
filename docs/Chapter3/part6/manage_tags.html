3.2.2 标签基本功能

如何不显示、删除和禁用自动标签，以及为文献添加标签。

  1. 不显示自动标签

    当 Zotero 导入文献时，自动标签功能会将关键词和主题导入为标签，可能导致标签区域杂乱。以下是不显示自动标签的方法：

    1. 如下图所示，点击左下角漏斗图标打开标签管理区域。
    2. 如下图所示，取消勾选"显示自动标签"。
  2. 删除现有自动标签
    1. 如下图左上角"第一步"所示，点击"我的文库"。
    2. 如下图左下角"第二步所示"，点击漏斗图标。
    3. 如下图左下角"第三步"所示，勾选"删除此文献库中的自动标签"。
    4. 如下图所示，点击"ok"，删除现有自动标签。
  3. 禁用自动标签
    1. 点击菜单栏"Zotero"
    2. 点击"设置"
    3. 点击"常规"
    4. 在"基于关键词和主题词自动为条目添加标签"前面方框内点击，使他处于不选中状态。
  4. 为文献添加新标签
    1. 如下图中上侧"第一步"所示，选中一篇文献。
    2. 如下图右侧"第二步"所示，点击标签图标。
    3. 如下图右上角"第三步"所示，点击加号图标。
    4. 如下图右侧"第四步"所示，在文本框里输入要新添加的标签名称，按回车键完成。
  5. 为多篇文献添加现有标签
    1. 如下图"第一步"所示，选中多篇文献。
    2. 如下图"第二步"所示，将选中的多篇文献拖动到左下角的标签区域内选中的现有标签处。
