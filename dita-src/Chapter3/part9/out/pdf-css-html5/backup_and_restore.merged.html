备份和恢复 Zotero 数据

Zotero 的备份包括设置和数据两个部分，以下内容介绍如何进行备份和恢复：

  • 设置：又称配置，包括插件及其配置、文件存储路径等。
  • 数据：包括文献条目、附件、样式文件和转换器等。

    需要注意的是，同步功能只能同步参考文献条目和附件（存放于 storage目录），但无法备份设置、样式文件等。建议手动或使用工具进行备份。

9.1 设置的备份与恢复

备份Zotero设置的方法，以及如何在新设备上恢复设置。

  1. 定位Zotero配置文件位置，以windows为例：
    Windows系统的默认路径为"C: Users <User Name> AppData Roaming Zotero Zotero Profiles <randomstring>"。Macos系统的默认路径为"/Users/<username>/Library/Application Support/Zotero/Profiles/<randomstring>"。
    1. 按Win+R，输入 %appdata%，然后回车，打开 Roaming目录。
    2. 再双击 Zotero 等文件夹进入最终的目录，如 vwadz9wo.default。
  2. 备份配置文件：
    1. 复制上述目录到安全位置，例如D: Zotero _back。
  3. 恢复配置文件：
    1. 按Win+R，输入<zotero安装路径>\zotero.exe -P。如D:\zotero\zotero.exe -P，并回车。
    2. 在弹出的对话框中选择 "Creat Profile"，并点击 Next。
    3. 输入配置文件名称，点击 Choose Folder...，并选择刚才备份的目录，如 D:Zotero_backvwadz9wo.default
    4. 点击 Finish，然后Start Zotero。
    5. 确保新设备与旧设备目录结构一致，恢复后的设置（包括插件和同步用户）会自动应用。

9.2 数据的备份与恢复

备份 Zotero 数据文件夹及重要文件的方法，以及如何恢复数据。

  1. 定位默认路径的Zotero 数据目录：
    1. Windows 系统默认路径为 " C: Users <User Name> Zotero"。Macos系统默认路径为"/Users/<username>/Zotero"。
  2. 定位非默认路径的Zotero数据目录：
    1. 点击: Edit-Settings-Advanced
    2. 点击：Show Data Directory Location。
  3. 备份数据文件夹：
    1. 复制数据目录到安全位置，例如 "D:\Zotero_data_backup"。
  4. 恢复数据文件夹：
    1. 在 Zotero 中依次点击 Edit -Settings -Advanced-Files and Folders。
    2. 点击 Data Directory Location下的 Choose...。
    3. 选择自己备份的目录并根据提示重新启动 Zotero即可打开以前的数据。
