5 笔记管理

5.1 添加笔记

5.1.1 高亮

高亮

高亮文本是快速标记文献重点的常用方法，便于后续整理。

在 PDF 文档中添加高亮或下划线

  1. 打开 PDF 文档。
  2. 选中目标句子，点击左侧图标为其高亮，或点击右侧图标为其添加下划线。
    选中的句子已成功高亮或添加下划线。

5.1.2 添加注释

添加注释

注释功能支持多彩标注，帮助高效记录阅读感想或学术见解。

在 PDF 文档中添加注释

  1. 在界面上部选择“添加笔记”图标。
  2. 点击颜色块选择注释颜色。
  3. 单击目标位置，键入笔记内容。
  4. 单击任意空白处以关闭对话框。

5.1.3 删除注释

删除注释

删除不再需要的标注，保持文档整洁。

删除注释

  1. 选中注释图标。
  2. 点击右侧“三点菜单”，选择“删除”。
  3. 或在选中注释图标后按键盘 ⌫ 删除键。

5.1.4 通过注释添加笔记

通过注释添加笔记

自动生成基于注释的笔记，并支持格式自定义。

通过注释添加笔记

  1. 打开 PDF 文件，右键点击，选择“通过注释添加笔记”。
  2. 自定义笔记内容格式。

5.1.5 添加独立笔记

添加独立笔记

独立笔记适合记录与具体文献条目无关的内容，便于学术管理。

添加独立笔记

点击 Zotero 工具栏上的“新建独立笔记”图标。
成功创建一个新的独立笔记。

独立笔记

独立笔记和子笔记在本质上是一样的，只不过独立笔记不属于任何一个条目，独立笔记显示时和条目是一个级别。

5.2 管理笔记

5.2.1 关联文献

关联文献

笔记间可建立条目关联，快速定位相关文献。

关联文献

  1. 点击“关联文献”栏中的“+”按钮。
  2. 确定关联文献，点击“选择”。
  3. 添加关联条目后，单击关联的条目就会跳转到该关联条目所在的位置。

关联逻辑

关联关系是双向的，但不具备传递性即条目A与B关联，Zotero会自动让B关联A；若A关联B，B关联C，A并不会自动关联C。。
