2.2.1 Zotero在浏览器中的插件安装

为了高效地导入文献，建议安装 Zotero 的浏览器扩展程序。Zotero 提供了适用于多种浏览器的扩展，包括 Chrome、Firefox、Safari 以及基于 Chromium 内核的最新版 Edge 浏览器，但不支持 Internet Explorer。

a 在Chrome中安装Zotero的浏览器插件

  1. 访问https://www.zotero.org/download/，网站会检测到浏览器为Chrome并提示安装相应插件。
  2. 点击Install Chrome Connector，然后会出现是否需要安装的提示。
  3. 点击添加扩展程序即可进行插件安装。

b 无法在Chrome中安装Zotero的浏览器插件

Condition

无法正常安装Chrome浏览器所使用的插件

Cause

未使用/无法使用VPN

Remedy

Users

  1. 选择离线下载：打开浏览器，搜索可以到https://crxdown.com/，输入 IDekhagklcjbdpajgpjgmbionohlpdbjgc，点击下载针对 Chrome 内核的插件，等待离线安装。
  2. 更换浏览器：打开Edge浏览器，访问https://www.zotero.org/download/，下载并安装。

2.2.2 Zotero的浏览器插件使用

下载并安装好与浏览器适配的插件后，即可开始使用。

a 在Chrome中使用Zotero的浏览器插件

在Chrome浏览器中使用插件保存搜索到的文献资料。

  1. 点击浏览器界面右上角拓展程序标志。
  2. 点击Zotero插件标志。
  3. 保存至Zotero中想保存位置，如“我的文库。
  4. 之后可以在Zotero相应位置找到被保存的资料.

b 无法在Chrome中使用Zotero的浏览器插件

Condition

无法在Zotero中找到自己曾操作保存的资料。

Cause

  1. Zotero插件不支持该网页内容格式。
  2. Zotero软件后台未启动。
  3. Zotero软件或其浏览器插件未更新至最新版本。

Remedy

Users

  1. 直接点击拓展程序标志，出现灰色Zotero标志，则表示该界面不支持被Zotero插件保存。
  2. 开始抓取资料前，打开Zotero软件，并保证其在后台不能退出。
  3. 更新Zotero软件（或/及）浏览器插件至最新版本，更新步骤与下载步骤相同。

2.2.3 Zotero插件下载与安装

Zotero 的插件功能是其强大特性之一，它允许用户通过添加插件来扩展 Zotero 的能力。这些插件通常具有 .xpi 文件扩展名，并且可以在开源平台如 GitHub 或 Gitee 上找到。本章将以Windows系统为例，围绕如何对Zotero插件进行下载与安装。

a 以安装“Zotero插件市场”插件为例

  1. 打开浏览器，在搜索框内键入https://github.com/syt2/zotero-addons/releases/latest/download/zotero-addons.xpi
    浏览器将下载Zotero插件市场的安装文件，使用者将其保存至自己选择的位置即可。
  2. 打开Zotero，在依次点击工具与插件，在弹出的对话框中点击齿轮图标选择Install Plugin From File。
  3. 弹出如图所示对话框后，找到刚才下载的插件市场的安装文件并点击，然后文件就会安装，待安装完毕后在管理插件界面就会出现刚才安装的“Zotero插件市场”插件。

b 通过Zotero插件市场安装插件（以“Jasminum”插件为例）

一些常用Zotero插件可通过Zotero插件市场安装，以下将以按装“Jasminum”插件为例。

  1. 点击Zotero工具条上Zotero插件市场，将会弹出Zotero插件市场界面，其上显示了Zotero插件市场支持的插件，内容包括插件名称、简介、发布信息、服务器最新版本、本地安装版本、本地装状态，此处建议将Automatic update add-ons选中，保证插件后期自动更新。
  2. 找到所需插件并选中，如Jasminum插件，再点击install，接下来Zotero插件市场将会安装该选中插件。
